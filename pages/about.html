關於Vite & GitHub使用步驟

  • 要注意vite.config.js裡面有個repo的name要與GitHub網站的Repo名字相同
  • 最一開始是 npm install 安裝
  • 然後 npm run dev 是跑起來環境，會開一個瀏覽器，會進入vite模式下，這時候可以做修改跟即時看結果
  • 都做完了ctrl+c離開Vite模式
  • 然後執行git的指令 (main:主要開發分支環境)
    1. git init
    2. git add .
    3. git commit -m "備註說明等等"
    4. git remote add origin \[GitHub Repositories Url]\ 例如 https://github.com/Josietaknow/ViteTest0724.git
    5. git branch -M main
    6. git push -u origin main
  • 最終要把結果發布到gh-pages就是執行 npm run deploy
  • 之後如果還有再繼續更新等等，就執行
    1. git add .
    2. git commit -m "備註說明等等"
    3. git push
  • 但是也是要記得同時npm run deploy
  • 之後在打開VS code要注意黃底的要再做一次
